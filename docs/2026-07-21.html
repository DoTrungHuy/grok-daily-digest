02 工具更新

Claude Code 推出屏幕阅读器模式，提升无障碍体验

"Claude Code now has a screen reader mode. Running claude --ax-screen-reader swaps the visual terminal UI for plain, linear text that screen readers (like VoiceOver and NVDA) can follow." — @unknown

实用点

对依赖辅助技术的开发者是重大利好。可通过 claude --ax-screen-reader 临时启用，或在 ~/.claude/settings.json 中设置 "axScreenReader": true 持久开启。支持带标签的行（you:、claude:、tool:）、数字菜单和提示音。

04 行业趋势

Google 在印度 IO Connect 展示 WebMCP 落地案例

"At #GoogleIOConnect India we asked Dhananjay Kumar, the VP of Engineering at @MakeMyTrip, how they are putting WebMCP to work." — @unknown

实用点

WebMCP（Web Model Context Protocol）是 Google 推动的 agent 与网页交互协议。MakeMyTrip 作为大型旅游平台采用该技术，表明 WebMCP 在真实业务场景中已有落地。另有开发者制作了交互式演示站点，可学习如何让 agent 操作网页。
